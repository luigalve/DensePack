DensePack

Created to cut input token cost of raw text.

Paste or drop any text: Code, markdown, JSON, prose. DensePack flattens it into one continuous block and renders tightly-cropped PNGs, cutting input token cost versus pasting the raw text. Tune font and size, watch the savings meter, and download.
